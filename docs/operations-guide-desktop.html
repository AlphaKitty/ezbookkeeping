一登录与注册

桌面端
🖥 登录与注册

打开网址后进入登录页面，左侧是品牌展示，右侧是登录表单。

  1. 在「用户名或邮箱」输入框中输入账号或邮箱地址。
  2. 在「密码」输入框中输入密码。
  3. 点击「登录」按钮进入系统。
  4. 如果没有账号，点击下方的「注册」链接，填写邮箱、用户名和密码即可创建账号。
  5. 忘记密码时，点击「忘记密码」，输入注册邮箱后会收到重置邮件。

二设置

桌面端
🖥 个人信息
  1. 点击右上角头像，选择「用户设置」。
  2. 在「基本信息」标签页可修改昵称、邮箱、默认货币、时区、语言、日期和数字格式。
  3. 在「安全」标签页可修改登录密码。
  4. 在「数据管理」标签页可查看数据统计、导出数据或清空数据。
🖥 应用设置
  1. 点击右上角头像，选择「应用设置」。
  2. 主题：切换浅色/深色/跟随系统。
  3. 应用锁：设置 PIN 码或指纹锁，打开应用需要验证。
  4. 统计设置：调整图表默认显示偏好。
  5. 浏览器缓存管理：管理离线数据缓存。

三账户管理

桌面端
🖥 账户管理
  1. 左侧导航栏点击「账户」进入账户管理页面。
  2. 左侧显示各类账户（现金、银行卡、信用卡等）的汇总余额，点击切换查看。
  3. 点击「添加账户」按钮，填写账户名称、选择币种、设置图标和颜色，还可以设置父账户创建子账户层级。
  4. 账户列表显示每个账户的名称、图标和当前余额。
  5. 点击账户旁的编辑按钮可修改账户信息，点击删除可移除账户。
  6. 支持对账：选中账户，点击对账按钮，逐条核对交易记录。
  7. 支持隐藏账户：不常用的账户可以隐藏，不在首页和列表中显示。
  8. 支持调整排序：按使用频率自由拖拽排序。

四分类与标签

桌面端
🖥 分类管理
  1. 左侧导航栏点击「分类」进入分类管理页面。
  2. 左侧顶部切换收支类型（支出/收入/转账），下方列出所有一级分类。
  3. 点击一级分类，右侧显示其下的二级分类列表。
  4. 点击「添加」按钮创建新分类：输入名称、选择图标和颜色、指定父分类。
  5. 支持预设模板：内置常用分类模板（餐饮、交通、购物等），可一键导入。
  6. 支持隐藏分类和调整排序。
🖥 标签管理
  1. 左侧导航栏点击「标签」进入标签管理页面。
  2. 左侧显示标签组（可创建如「客户」「项目」「员工」等分组）。
  3. 点击标签旁的编辑按钮可内联编辑标签名称，直接在当前行修改。
  4. 支持批量添加标签和隐藏不常用标签。

五记一笔账

桌面端
🖥 记一笔支出
  1. 点击顶部工具栏的「记一笔」按钮，弹出记账对话框。
  2. 左侧竖排选择交易类型：「支出」。
  3. 在金额框中输入数字，比如 150.00。支持算式，输入 30*5 自动算出 150.00。
  4. 点击分类选择器，先选一级分类（如「餐饮食品」），再选二级分类（如「午餐」）。
  5. 选择扣款账户，比如「工商银行卡」。
  6. 确认日期和时间，默认是当前时间，可按需修改。
  7. （选填）在备注框中输入文字说明，比如「请客户吃饭」。
  8. （选填）添加标签，如「招待」，方便后续按标签筛选。
  9. （选填）切换到「图片」标签页上传收据或发票照片。
  10. 点击底部「保存」按钮完成记账。
左侧「基本信息」「地图位置」「图片」三个标签页可以切换表单内容。
🖥 记一笔收入

操作与支出相同，区别在于左侧选择「收入」类型，分类和金额按收入来处理。

🖥 记一笔转账
  1. 左侧选择「转账」类型。
  2. 输入转出金额。
  3. 选择「来源账户」（转出）和「目标账户」（转入），比如从「工商银行卡」转 5000 元到「支付宝」。
  4. 如果两个账户币种不同，系统会自动显示汇率换算结果。
  5. 确认日期和分类后点击「保存」。

六导入账单

桌面端
🖥 导入账单
  1. 在账单列表页面，点击顶部工具栏的「导入」按钮。
  2. 在弹出的对话框中选择文件来源类型（如支付宝、微信、银行 OFX/QIF/CSV 等）。
  3. 选择要导入的文件，系统会自动解析并预览数据。
  4. 确认数据无误后，点击「导入」完成。
支持格式：银行流水（OFX/QFX/QIF/IIF/MT940/Camt.052/Camt.053/CSV）、支付宝 CSV、微信 CSV/XLSX、京东金融 CSV、随手记 XLS/XLSX、GnuCash XML、Firefly III CSV、Beancount、Excel 等。

七首页看板 —— 打开就能了解经营状况

桌面端
🖥 首页看板

登录后默认进入首页看板，所有关键数据集中展示在一个页面中。

  1. 左侧总览区：显示本月支出金额和净资产总额。点击眼睛图标可以隐藏/显示金额数字。
  2. 右上资产区：列出所有账户及其余额，一眼看清每个账户里有多少钱。
  3. 趋势图表：展示本月每日收支趋势，柱状图直观对比每天的收入和支出。
  4. 快捷日期卡片：今日、本周、本月、本年的收支汇总，点击任意卡片跳转到对应时间段的账单列表。
左侧是常驻导航栏，可以随时切换到账单、统计、账户等其他功能。

八查看和搜索账单

桌面端
🖥 查看和搜索账单
  1. 左侧导航栏点击「账单」，进入账单列表页面。
  2. 页面顶部可切换三种视图：列表（表格形式）、图片（卡片式预览）、日历（按日期分布）。
  3. 左侧筛选区可以按交易类型（全部/收入/支出/转账）、时间范围（如本月、上月、自定义日期）快速筛选。
  4. 顶部搜索框输入关键字（备注内容、金额等），回车或点击搜索图标即可过滤。
  5. 每笔账单右侧有编辑和删除按钮，点击编辑可修改该笔账单的所有信息。
  6. 勾选多笔账单后，可以使用顶部的「批量操作」统一修改分类、账户或标签。
  7. 每页显示数量可通过左侧下拉框调整（20/50/100 条）。

九定期记账与模板

桌面端
🖥 定期交易

定期交易适合每月固定的收支项目（如租金、工资、贷款还款），设置一次后系统自动按时记账。

  1. 左侧导航栏点击「定期交易」进入管理页面。
  2. 点击「添加」，填写交易信息（类型、金额、分类、账户等），和记一笔普通账一样。
  3. 填写模板名称（如「每月房租」）。
  4. 设置执行频率：每小时、每天、每周、每月、每年，或自定义 Cron 表达式。
  5. 设置开始日期和结束日期（可选）。
  6. 保存后系统会按设定频率自动生成交易记录。
🖥 模板

模板是预设好的交易格式，适合频繁发生的同类交易，一键套用。

  1. 左侧导航栏点击「模板」进入模板管理页面。
  2. 创建模板时填写常用的交易信息（类型、金额、分类、账户等），给模板命名。
  3. 记账时在对话框中点击「从模板创建」，选择一个模板即可自动填充所有字段。

十对账

桌面端
🖥 对账

对账是将系统中的交易记录与银行账单逐条核对，确保两边数据一致。

  1. 在账户管理页面，选中要对账的账户。
  2. 点击「对账」按钮，进入对账页面。左侧显示该账户的所有交易记录。
  3. 逐条勾选与银行账单匹配的交易，已勾选的金额会自动累加。
  4. 对完当前所有交易后，可以点击「更新对账时间」，记录本次对账的时间点。
  5. 以后对账时可以筛选「上次对账时间之后」的交易，只核对新增部分。

十一统计分析

桌面端
🖥 统计分析
  1. 左侧导航栏点击「统计」进入统计分析页面。
  2. 左侧面板可选择分析类型：
    • 分类统计：饼图展示各分类的收支占比。
    • 趋势分析：折线图/柱状图展示收支随时间的变化。
    • 资产趋势：各账户余额的长期变化趋势。
  3. 选择图表类型：饼图、柱状图、折线图、雷达图、热力图、桑基图、层级图等。
  4. 左侧下方可切换数据维度：总览、按账户、按分类、按标签等。
  5. 支持导出统计图表为图片或数据文件。
左侧面板所有筛选条件实时生效，切换维度无需刷新页面。

十二库存管理 即将上线

桌面端
🖥 物品建档
  1. 左侧导航栏点击「物品定义」进入物品管理页面。
  2. 点击「添加物品」，填写名称、单位（个/箱/kg 等）、分类和图标。
  3. 设置进价表达式和售价表达式，系统可根据公式自动计算金额。
  4. 所有物品统一建档，方便统一管理和查找。
🖥 进出库记录
  1. 左侧导航栏点击「库存记录」进入库存页面。
  2. 点击「添加记录」登记一次库存变动：选择物品、填写数量、单价、变动日期。
  3. 系统自动计算当前库存余量。
  4. 支持按物品、时间范围筛选查看库存变动历史。
🖥 交易关联库存
  1. 记账时，在对话框中勾选「关联库存」选项。
  2. 选择关联的物品和数量。
  3. 保存交易后，库存自动扣减对应数量，无需手动更新库存。
  4. 进价和售价分开管理，利润一目了然。
