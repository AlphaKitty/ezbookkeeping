一登录与注册

移动端
📱 登录与注册

打开网址后进入全屏登录页面，顶部显示应用名称，下方是表单区域。

  1. 在「用户名或邮箱」输入框中输入账号或邮箱地址。
  2. 在「密码」输入框中输入密码。
  3. 点击「登录」按钮进入系统。
  4. 点击底部「注册」链接可创建新账号，填写邮箱、用户名和密码。
  5. 点击「忘记密码」会从底部弹出重置面板，输入邮箱后发送重置邮件。
移动端登录页面底部有语言切换选项，可选择中文、英文等语言。

二设置

移动端
📱 设置

移动端所有设置集中在一个页面中，通过底部导航栏最右侧齿轮图标进入，向下滚动可查看所有选项。

  1. 个人设置区（顶部）：
    • 点击头像区域进入：个人信息修改、分类管理、标签管理、模板管理、物品管理、库存管理、数据管理、会话管理、退出登录。
  2. 应用设置区（下方）：
    • 主题：切换浅色/深色/跟随系统。
    • 字体大小：调整显示文字大小。
    • 时区：切换时区设置。
    • 应用锁：设置 PIN 码或指纹锁。
    • 汇率：查看和管理汇率数据。
    • 余额显示：开关首页金额显示。
    • 页面设置：自定义首页显示内容。
    • 统计设置：调整图表默认偏好。
    • 浏览器缓存：管理离线数据。
  3. 大部分设置项为开关或列表选择形式，点击即可更改。

三账户管理

移动端
📱 账户管理
  1. 点击底部导航栏第二个图标进入账户页面。
  2. 顶部卡片显示净资产总额（可点击眼睛图标隐藏），下方显示总资产和总负债。
  3. 账户按类别分组显示（现金、银行卡、信用卡等），每项显示名称、图标和余额。
  4. 点击右上角 + 按钮添加新账户。
  5. 左滑任意账户可快速编辑或删除。
  6. 点击右上角菜单按钮可以进行排序、调整显示顺序、管理隐藏账户。
  7. 点击某个账户进入详情页，可以查看该账户的交易流水和余额趋势。

四分类与标签

移动端
📱 分类管理
  1. 进入设置页面，点击「分类」进入分类管理。
  2. 顶部切换收支类型（支出/收入/转账）。
  3. 点击某个一级分类进入其二级分类列表。
  4. 点击右上角 + 按钮添加新分类。
  5. 左滑任意分类可快速编辑、隐藏或删除。
  6. 点击右上角菜单进行排序、分组管理、显示隐藏项等操作。
📱 标签管理
  1. 进入设置页面，点击「标签」进入标签管理。
  2. 顶部可切换标签组。
  3. 左滑标签可移动、编辑或删除。
  4. 点击标签直接内联编辑名称。
  5. 点击右上角菜单管理标签组和显示设置。

五记一笔账

移动端
📱 记一笔支出
  1. 点击底部导航栏中间的 + 按钮，进入记账页面。
  2. 顶部切换类型为「支出」。
  3. 点击金额区域，弹出数字键盘，输入金额如 150.00。
  4. 点击「分类」，从底部滑出的分类树中选择，先选一级再选二级。
  5. 点击「账户」，从列表中选择扣款账户。
  6. 日期时间默认当前，点击可修改。
  7. （选填）点击「备注」输入文字说明。
  8. （选填）点击「标签」选择或新建标签。
  9. （选填）点击「图片」区域的 + 号拍照或从相册上传收据。
  10. 点击右上角 ✓ 按钮或底部「保存」按钮完成记账。
在「更多」菜单中可以使用「复制」「粘贴金额」「切换账户方向」等快捷操作。
📱 记一笔收入

操作与支出相同，区别在于顶部选择「收入」类型，分类和金额按收入来处理。

📱 记一笔转账
  1. 顶部切换类型为「转账」。
  2. 输入转出金额，选择来源账户和目标账户。
  3. 如果两个账户币种不同，转出金额下方会显示汇率换算参考。
  4. 确认分类和日期后点击保存。

六导入账单

移动端
📱 导入账单
  1. 在账单列表页面，点击右上角菜单按钮。
  2. 选择「导入」。
  3. 选择文件来源类型（如支付宝、微信、银行 OFX/QIF/CSV 等）。
  4. 选择手机上的文件。
  5. 预览解析结果，确认后点击导入。
移动端同样支持桌面端的所有导入格式：银行流水、支付宝 CSV、微信 CSV/XLSX、京东金融 CSV、随手记 XLS/XLSX、GnuCash XML、Firefly III CSV、Beancount、Excel 等。

七首页看板 —— 打开就能了解经营状况

移动端
📱 首页看板

登录后进入首页，顶部是醒目的月度支出卡片，向下滚动查看更多信息。底部是五个固定的导航按钮。

  1. 顶部支出卡片：显示本月支出总额，点击眼睛图标可隐藏/显示金额。
  2. 时段汇总列表：依次显示今日、本周、本月、本年的收支金额，点击任意一行跳转到对应账单列表。
  3. 底部导航栏：
    • 账单（第一个图标）：查看所有交易记录。
    • 账户（第二个图标）：管理账户和余额。
    • + 按钮（中间大按钮）：记一笔账。
    • 统计（第四个图标）：查看图表分析。
    • 设置（第五个齿轮图标）：个人信息和应用设置。
  4. 下拉页面可以刷新数据。
长按底部中间的 + 按钮，可以快速使用拍照记账或模板记账。

八查看和搜索账单

移动端
📱 查看和搜索账单
  1. 点击底部导航栏第一个图标进入账单列表。
  2. 顶部标题栏点击可在「列表」「图片」「日历」三种视图间切换。
  3. 点击右上角搜索图标，顶部滑出搜索栏，输入关键字即可筛选。
  4. 底部工具栏提供多种筛选方式：
    • 点击日期范围切换时间段（本月、上月、近三个月等）。
    • 点击分类图标按分类筛选。
    • 点击账户图标按账户筛选。
    • 点击「更多」可按交易类型、金额范围、标签筛选。
  5. 账单列表向下滚动会自动加载更多记录。
  6. 点击任意一笔账单进入详情页，可以编辑或删除。
  7. 下拉页面可以刷新数据。

九定期记账与模板

移动端
📱 定期交易与模板

定期交易适合每月固定的收支项目（如租金、工资），设置一次后系统自动按时记账。模板适合频繁发生的同类交易，一键套用。

  1. 进入设置页面，点击「定期交易」或「模板」进入管理页面。
  2. 点击 + 按钮创建，填写交易信息（类型、金额、分类、账户等）。
  3. 创建定期交易时需设置执行频率和时间范围。
  4. 创建模板时需填写模板名称。
  5. 编辑和删除操作与桌面端一致。
  6. 记账时长按底部 + 按钮，可以快速选择模板记账。
在记账页面点击菜单中的「复制」功能，也可以快速复用之前的交易记录。

十对账

移动端
📱 对账

对账是将系统中的交易记录与银行账单逐条核对，确保两边数据一致。

  1. 在账户详情页中点击对账入口。
  2. 逐条勾选与银行账单匹配的交易。
  3. 查看核对结果，确认金额是否一致。

十一统计分析

移动端
📱 统计分析
  1. 点击底部导航栏第四个图标进入统计页面。
  2. 点击顶部标题可切换分析类型（分类统计 / 趋势分析 / 资产趋势）。
  3. 底部工具栏可切换图表类型（饼图、柱状图、折线图等）。
  4. 点击日期按钮切换时间范围。
  5. 点击「更多」按钮可按账户、分类、标签等维度细化筛选。
  6. 点击右上角菜单可以进入统计设置页面，调整图表显示偏好。

十二库存管理 即将上线

移动端
📱 物品建档
  1. 进入设置页面，点击「物品」进入物品管理。
  2. 点击 + 按钮添加物品：填写名称、单位（个/箱/kg 等）、分类和图标。
  3. 设置进价表达式和售价表达式，系统可根据公式自动计算金额。
📱 进出库记录
  1. 进入设置页面，点击「库存」进入库存记录管理。
  2. 点击 + 按钮登记库存变动：选择物品、填写数量、单价、变动日期。
  3. 系统自动计算当前库存余量。
  4. 支持按物品、时间范围筛选查看库存变动历史。
📱 交易关联库存
  1. 记账时在表单中开启「关联库存」选项。
  2. 选择关联的物品和数量。
  3. 保存交易后库存自动扣减，进价和售价分开管理。
